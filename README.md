# نمره شمار
همیشه جمع زدن نمرات یه برگه امتحان، مخصوصا زمانی که تعداد سوالات زیاده و بارم بندی بر حسب 0.25 است، کاری حوصله سربر بوده برای من. 
این شد که تصمیم گرفتنم این نمره شمار رو بنویسم تا زحمت جمع زدن نمرات رو واسم بکشه.
برای من خیلی کاربردیه، شاید به کار دبیران دیگه هم بیاد.
